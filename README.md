# Gestao-de-estoque
# 📦 Sistema de Gestão de Estoque

Sistema simples de controle de estoque desenvolvido em Python com SQLite.

## ✨ Funcionalidades
- Cadastro de produtos
- Atualização e exclusão
- Consulta de estoque
- Geração de relatórios
- Validações de entrada

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/m4teusCruz/gestao-de-estoque.git

# Entre na pasta
cd gestao-de-estoque

# Execute o programa
python src/main.py
